How To Join

01

Launch Minecraft

.

Open your Minecraft Java Edition launcher and start the game (Version 1.20+ recommended).

02

Multiplayer

.

Click on the "Multiplayer" button from the main menu.

03

Add Server

.

Click "Add Server" and enter the name "Bramayugam MC".

04

Enter IP

.

In the Server Address field, paste: bramayugam.qzz.io

05

Connect

.

Click "Done", select the server from your list, and click "Join Server". Welcome home!
